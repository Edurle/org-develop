# E2E Testing Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Comprehensive e2e tests covering all 58 API endpoints, 12 frontend pages, and all state transition paths using Playwright + pytest.

**Architecture:** Shared `ApiHelper` and `UiHelper` classes wrap all API/Playwright interactions. Each test file covers one domain module with both API-level and UI-level tests. Every test is data-isolated (registers its own user, creates its own resources).

**Tech Stack:** Python 3.11+, pytest, pytest-playwright, requests

## Running the Tests

```bash
# Start backend + frontend first
cd backend && uvicorn app.main:app --host 0.0.0.0 --port 8000 &
cd frontend && npm run dev &

# Run all e2e tests
cd tests && pytest -v

# Run single module
cd tests && pytest test_auth.py -v

# Run only API tests (skip Playwright)
cd tests && pytest -v -k "not UI"

# Run only UI tests
cd tests && pytest -v -k "UI"
```
