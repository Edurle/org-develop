# 前端国际化（i18n）设计

## 背景

前端所有用户可见文本硬编码为英文，需要支持中英文切换。

## 方案

使用 `vue-i18n` + JSON 翻译文件。

### 文件结构

```
src/locales/
  en.json        # 英文翻译
  zh-CN.json     # 简体中文翻译
  index.ts       # i18n 实例 + 语言检测 + 切换逻辑
```

### 依赖

- `vue-i18n` ^10.x（Vue 3 官方 i18n 库）

### 翻译 key 组织

按页面/功能分命名空间：

```json
{
  "common": { "save": "Save", "cancel": "Cancel", "delete": "Delete", "edit": "Edit", "loading": "Loading...", "actions": "Actions", "confirm": "Confirm" },
  "auth": { "signIn": "Sign in", "createAccount": "Create account", "logout": "Logout", "username": "Username", ... },
  "project": { "title": "Projects", "newProject": "New Project", ... },
  "requirement": { "title": "Requirements", "newRequirement": "New Requirement", ... },
  "specification": { "title": "Specifications", "createSpec": "Create Spec", ... },
  "task": { "devTasks": "Dev Tasks", "testTasks": "Test Tasks", ... },
  "testcase": { "title": "Test Cases", ... },
  "coverage": { "title": "Coverage", ... },
  "status": { "draft": "Draft", "open": "Open", "inProgress": "In Progress", ... },
  "priority": { "low": "Low", "medium": "Medium", "high": "High", "critical": "Critical" }
}
```

### 集成方式

- `main.ts` 注册 i18n 插件
- 模板用 `$t('key')`
- 脚本用 `const { t } = useI18n()`

### 语言检测与切换

- 默认英文
- 首次访问检测 `navigator.language`，匹配到 zh-CN 则自动切换
- 手动切换后存入 localStorage
- AppLayout 导航栏添加语言切换按钮（EN / 中文）

### 不做的事

- 不做日期/数字本地化（已有 toLocaleDateString）
- 不翻译后端 API 返回的错误消息
- 不做 URL 本地化
